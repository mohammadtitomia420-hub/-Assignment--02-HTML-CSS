# Foodies — Responsive Food Website (HTML & CSS)

This project is a simple food-related landing page built using pure HTML and CSS. It includes a hero section, featured food items, responsive layout, and a clean modern UI. Perfect practice for beginner frontend development.

## Features
✅ Fully responsive layout
✅ Hero banner with call-to-action
✅ Food cards/menu section
✅ Footer section with copyright
✅ Clean and organized code

## Technologies Used
- HTML5
- CSS3 (Flexbox / Responsive)

## How to Run
Just open `index.html` in any browser.

## Purpose
This project was created as an assignment to practice layout design, responsiveness, and CSS styling.
